# Esercitazioni in React

## Informazioni utili
Questi progetti sono presi dal corso di Udemy, "React e Redux in italiano", e sono stati usati come esercitazione. <br>
Lo stile e il progetto è gia preimpostato, la logica invece verrà implementata da me.

## come installare e visualizzare i progetti
Una volta scaricata la cartella come zip, o clonata la repository,
apritela nell'IDE e seguite i seguenti comandi: 

- spostarsi nella cartella inizio di ogni progetto: <br>
cd nomeProgetto/inizio <br>
per esempio:
```
cd appuntamenti/inizio
```

- installare le dipendenze:
```
npm i
```

- avviare il progetto sul browser:
```
npm run start
```

- collegarsi all'url del progetto sul browser che di solito è **http://localhost:3000**
